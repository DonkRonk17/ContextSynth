# ContextSynth Branding Prompts

## DALL-E Image Generation Prompts

Following **Beacon HQ Visual System v1** style guide.

---

## Style Guidelines

**Color Palette:**
- Primary Background: Dark navy (#0a0e27)
- Primary Accent: Bright cyan (#00d4ff)
- Secondary: Electric purple (#7c3aed)
- Success: Emerald (#10b981)

**Theme:** Information synthesis, context aggregation, knowledge distillation

---

## Prompt 1: Title Card (16:9 Landscape)

```
A professional tech-themed title card for "ContextSynth" software tool.

Dark navy blue background (#0a0e27) with subtle data stream patterns.

Central focal point: A stylized brain or neural network icon merging with document/file icons, creating a synthesis effect. Bright cyan (#00d4ff) energy flows connect multiple source documents into a single glowing summary.

Below the icon, abstract representations of code snippets, JSON data, and markdown documents flowing into a central point.

Text "CONTEXTSYNTH" in bold, modern sans-serif font below, with cyan glow effect.

Tagline: "Instant context. Zero friction." in smaller white text.

Floating code symbols (function names, imports, TODOs) fading into the background.

Tech circuit patterns along edges. Clean, professional, futuristic aesthetic.

16:9 aspect ratio. No text spelling errors.
```

## Prompt 2: Logo Mark (1:1 Square)

```
A minimalist square logo for "ContextSynth" software tool.

Dark navy blue background (#0a0e27).

Central icon: A stylized open book or document with neural network nodes overlaid. The nodes connect in a synthesis pattern, converging to a single bright cyan (#00d4ff) point at the center.

Clean geometric lines suggesting information extraction and aggregation.

Subtle glow effect around the central convergence point.

Modern, minimalist design. Professional and tech-forward.

1:1 aspect ratio. No text, icon only.
```

## Prompt 3: Horizontal Logo (3:1 Wide)

```
A horizontal logo banner for "ContextSynth" software tool.

Dark navy blue background (#0a0e27).

Left side: Document/synthesis icon with converging information streams, glowing bright cyan (#00d4ff).

Right side: "CONTEXTSYNTH" text in bold, clean, modern sans-serif font. White text with subtle cyan glow on "CONTEXT" and "SYNTH".

Connecting element: A flowing data stream between icon and text representing synthesized information.

Professional tech aesthetic. Clean and modern.

3:1 aspect ratio. Suitable for GitHub banner or header.
```

## Prompt 4: App Icon (1:1 Simplified)

```
A simplified app icon for "ContextSynth" software tool.

Dark navy blue circular background (#0a0e27) with subtle gradient.

Central element: A minimal document icon with 3 converging lines/arrows pointing to a central dot. Bright cyan (#00d4ff) for the convergence point, white/gray for the document outline.

Clean, recognizable at small sizes. Modern flat design with subtle glow.

No text. Icon only.

1:1 aspect ratio. Suitable for 64x64px to 512x512px app icons.
```

## Usage Notes

1. **Title Card**: README headers, documentation, presentations
2. **Logo Mark**: Avatars, favicons, small displays
3. **Horizontal Logo**: GitHub repo banner, tool headers
4. **App Icon**: Application icons, shortcuts, taskbar
